# 🖥 CPU Performance Analysis Tool

A Computer Architecture simulation project developed in Python that models fundamental CPU operations including the Fetch-Decode-Execute cycle, cache memory management, branch prediction, ALU execution, and performance analysis.

The simulator executes a set of instructions, tracks CPU performance metrics, and generates graphical reports to visualize system behavior.

## Features

### CPU Simulation

* Fetch-Decode-Execute Cycle
* Register Management
* Memory Operations
* Instruction Processing

### ALU Operations

* Addition (ADD)
* Subtraction (SUB)
* Multiplication (MUL)
* AND Logic Operation
* OR Logic Operation
* XOR Logic Operation

### Cache Simulation

* Direct-Mapped Cache
* Cache Hit Tracking
* Cache Miss Tracking
* Cache Replacement Mechanism
* Memory Access Performance Analysis

### Branch Prediction

* Branch Prediction Simulation
* Correct Prediction Tracking
* Misprediction Detection
* Branch Accuracy Analysis

### Performance Visualization

* Execution Time Analysis
* Cache Hit vs Miss Ratio
* Branch Prediction Accuracy
* Instruction Cycle Consumption Charts
